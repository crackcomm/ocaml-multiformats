Module B32

val encode : Core.Bigstring.t -> Core.Bigstring.t

encode alphabet data encodes data using alphabet defined in RFC 4648.

val encode_case : upper:bool -> Core.Bigstring.t -> Core.Bigstring.t

encode_case upper alphabet data encodes data using alphabet defined in RFC 4648. If upper is true the output is in uppercase alphabet.

val encode_lower : Core.Bigstring.t -> Core.Bigstring.t

encode_lower alphabet data encodes data using lowercase alphabet defined in RFC 4648.

val encode_upper : Core.Bigstring.t -> Core.Bigstring.t

encode_upper alphabet data encodes data using uppercase alphabet defined in RFC 4648.

val decode : Core.Bigstring.t -> Core.Bigstring.t

decodedata decodes data using alphabet defined in RFC 4648.
